🔥 Fix Firestore Rules

Nếu kết nối thất bại, Security Rules thường đã hết hạn (30 ngày).

1. Vào console.firebase.google.com
2. Chọn project pythontest-393209 → Firestore Database → tab Rules
3. Dán vào và nhấn Publish:

rules_version = '2';
service cloud.firestore {
 match /databases/{database}/documents {
  match /{document=**} {
   allow read, write: if true;
  }
 }
}
